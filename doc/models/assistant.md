
# Assistant

Semgrep Assistant data. Only present if Assistant is enabled

*This model accepts additional fields of type object.*

## Structure

`Assistant`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Autofix` | [`Autofix`](../../doc/models/autofix.md) | Optional | Fix data generated by Semgrep Assistant |
| `Autotriage` | [`Autotriage`](../../doc/models/autotriage.md) | Optional | Triage recommendation generated by Semgrep Assistant |
| `Component` | [`Component`](../../doc/models/component.md) | Optional | Semgrep Assistant's guess as for what the matched source code's purpose is |
| `Guidance` | [`Guidance`](../../doc/models/guidance.md) | Optional | Remediation guidance generated by Semgrep Assistant |
| `RuleExplanation` | [`RuleExplanation`](../../doc/models/rule-explanation.md) | Optional | AI-generated explanation of why a rule flagged this specific code |
| `AdditionalProperties` | `object this[string key]` | Optional | - |
