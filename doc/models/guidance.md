
# Guidance

Remediation guidance generated by Semgrep Assistant

*This model accepts additional fields of type object.*

## Structure

`Guidance`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Instructions` | `string` | Optional | Step-by-step instructions explaining to a developer how to fix the finding. AI generated content, review carefully |
| `Summary` | `string` | Optional | Short title explaining to a developer how to fix the finding. AI generated content, review carefully |
| `AdditionalProperties` | `object this[string key]` | Optional | - |

## Example

```csharp
using SemgrepWebApp.Standard.Models;
using SemgrepWebApp.Standard.Utilities;

Guidance guidance = new Guidance
{
    Instructions = "instructions4",
    Summary = "Use a template rendering engine such as EJS instead of string concatenation.",
    ["exampleAdditionalProperty"] = ApiHelper.JsonDeserialize<object>("{\"key1\":\"val1\",\"key2\":\"val2\"}"),
};
```
