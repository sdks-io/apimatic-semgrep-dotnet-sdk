
# Autotriage

Triage recommendation generated by Semgrep Assistant

*This model accepts additional fields of type object.*

## Structure

`Autotriage`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Reason` | `string` | Optional | The reasoning for a `false_positive` verdict; this explains why you might want to ignore the finding. Empty string if verdict is `true_positive` |
| `Verdict` | [`Verdict1?`](../../doc/models/verdict-1.md) | Optional | The verdict is `true_positive` if Assistant recommends fixing, `false_positive` if Assistant recommends ignoring this finding. AI generated decision, review carefully |
| `AdditionalProperties` | `object this[string key]` | Optional | - |

## Example

```csharp
using SemgrepWebApp.Standard.Models;
using SemgrepWebApp.Standard.Utilities;

Autotriage autotriage = new Autotriage
{
    Reason = "The matched code is used for a non-security related feature.",
    Verdict = Verdict1.FalsePositive,
    ["exampleAdditionalProperty"] = ApiHelper.JsonDeserialize<object>("{\"key1\":\"val1\",\"key2\":\"val2\"}"),
};
```
