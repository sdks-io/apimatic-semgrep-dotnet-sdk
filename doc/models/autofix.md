
# Autofix

Fix data generated by Semgrep Assistant

*This model accepts additional fields of type object.*

## Structure

`Autofix`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `Explanation` | `string` | Optional | DEPRECATED: This field is deprecated and will always be an empty string. Find a description of how this fix works under `assistant.guidance` |
| `FixCode` | `string` | Optional | Source code that replaces all matched lines to fix this finding. AI generated content, review carefully |
| `AdditionalProperties` | `object this[string key]` | Optional | - |

## Example

```csharp
using SemgrepWebApp.Standard.Models;
using SemgrepWebApp.Standard.Utilities;

Autofix autofix = new Autofix
{
    Explanation = "explanation2",
    FixCode = "cookie.setHttpOnly(true);\\nresponse.addCookie(cookie);",
    ["exampleAdditionalProperty"] = ApiHelper.JsonDeserialize<object>("{\"key1\":\"val1\",\"key2\":\"val2\"}"),
};
```
